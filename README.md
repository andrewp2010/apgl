# APGL
APettit OpenGL support library and test framework

Configure and build with CMake using repository scripts:
- Windows Batch Commands:
    ```
        .\scripts\configure.bat
        .\scripts\build.bat
        .\scripts\run.bat
    ```
## Style Guide
General:
- Acronyms of any size are considered a single word

snake_case:
- Directory names

camelCase:
- Namespaces
- Variables
- Functions/methods

PascalCase
- File names
- Classes
- Interfaces
